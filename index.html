log.js

This Javascript library aids development in an easy way. Small efforts with a lot of possibilities. Logging with log.js may be done at any point in any structure.


Download

Give it a try

Use the controls below to give log.js a try.
Open up your console, write your prefix and/or message and press enter.

/*
  Enter a message and press enter.
  Optional prefixes:
  x - System
  c - Critical
  e - Error
  w - Warning
  i - Info
  d - Debug
*/
log("");

Implementation

As many other libraries for Javascript; log.js is easily implemented with just one line of markup.

<script src="js/log-min.js"></script>

Easy as pie

To perform a log message, just use the log method from anywhere you want.

log("Hello World!");	// output: log.js - Hello World

Put your label on it

You may set a personal label on the messages by setting the logging.title string.

logging.title = "Google";

log("d User performed search");	// output: Google#DEBUG - User performed search
